local-secrets

A local-only secret and shared-file server for development across multiple git worktrees.

One variable per line: ENV_VAR_NAME=value. The backing file is kept at {{env_path}} with chmod 600; the storage directory is {{root_path}}.

Optional: store local untracked files once and fetch them from any worktree. File names may contain letters, numbers, dots, dashes, and underscores.
