Quick API: how do I use this?

local-secrets stores a flat set of keys under ~/.config/local-secrets/. Each key is a self-contained namespace with its own env file and shared files. Switch the active key with the dropdown on the main page.

Docker run

docker run --env-file <(curl -fsS {{base}}/env-file) image

Docker Compose

Keep Compose generic and pass local-secrets as a normal env-file path.

services:
  app:
    image: your-image:dev
    env_file:
      - ${DEV_ENV_FILE:?set DEV_ENV_FILE}
DEV_ENV_FILE=<(curl -fsS {{base}}/env-file) docker compose up

Fish shell:

env DEV_ENV_FILE=(curl -fsS {{base}}/env-file | psub) docker compose up

POSIX shell (bash/zsh/sh)

eval "$(curl -fsS {{base}}/export)"

Fish shell

curl -fsS {{base}}/export/fish | source

JSON

curl -fsS {{base}}/env

Shared local files

curl --data-binary @.npmrc -X PUT {{base}}/files/.npmrc
curl -fsS {{base}}/files/.npmrc
